## TODO APP  

Simples TODO APP utilizando MEAN Stack :D

==============================================

### PRÉ-REQUISITOS
  . MongoDB Instalado
  . NodeJS Instalado

===============================================

### TO RUN
  . Clone este repositório
  
  . Entre no mesmo e digite:
    ```
      npm install
    ```
  
  . Em seguida digite:
    ```
      npm start
    ```
  
  . Abra o navegador e acesse a página em **http://localhost:4000**

================================================

#### E É ISTO :D
