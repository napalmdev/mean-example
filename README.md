## TODO APP  

Simples TODO APP utilizando MEAN Stack :D

### PRÉ-REQUISITOS
  . MongoDB Instalado
  . NodeJS Instalado


## RUN
  . Clone este repositório
  . Entre no mesmo e digite:
    ```shell
      npm install
    ```
  . Em seguida digite:
    ```shell
      npm start
    ```
  . Abra o navegador e acesse a página em **http://localhost:4000**


## E É ISTO :D
